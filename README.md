# RTSR
## Real-Time Software Rendering  
![](https://github.com/IbrahimHindawi/rtsr/blob/main/rtsr.png)  

Windows:  
-`bbuild.bat`  
-`brun.bat`  

Apple:  
-`make`  
-`make run`  

Linux:  
-`make`  
-`make run`  

Dependencies:  
UNIX-based: sdl2 library  
WIN-based: visual studio 19 + vcpkg
